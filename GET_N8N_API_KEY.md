# 🔑 如何取得 n8n API Key

## 📍 您現在在這裡
```
https://n88.zeabur.app/home/workflows
```

---

## 🎯 取得 API Key 步驟（圖文說明）

### 步驟 1：進入設定頁面

1. **看向畫面右上角**，找到您的使用者圖示/頭像
   ```
   [您的頭像圖示] ← 點這裡
   ```

2. 會出現下拉選單，點擊 **"Settings"** (設定)

---

### 步驟 2：進入 API 設定

1. 在左側選單中找到 **"API"** 選項
   ```
   左側選單：
   - Personal
   - Security
   - API ← 點這裡
   - ...
   ```

2. 點擊進入 API 設定頁面

---

### 步驟 3：創建 API Key

1. 在 API 頁面中，找到 **"Create API Key"** 按鈕

2. 點擊後會生成一個新的 API Key

3. **立即複製這個 Key**（只會顯示一次！）
   - 格式：`n8n_api_xxxxxxxxxxxxxxxxxxxxxxxxxx`
   - 長度：約 35-40 個字元

4. 把這個 Key 貼給我

---

## ⚡ 快速路徑

如果找不到按鈕，試試這個直接連結：
```
https://n88.zeabur.app/settings/api
```

## ⚠️ 重要提醒

- **只會顯示一次**：API Key 創建後只顯示一次，請立即複製
- **安全保管**：這個 Key 可以完全控制您的 n8n
- **用完可刪除**：稍後可以在同一頁面刪除這個 Key

---

## ❓ 遇到問題？

### 問題 1：找不到「Settings」選項
**可能原因**：版本差異或權限問題
**解決方案**：
- 試試點擊左下角的齒輪圖示 ⚙️
- 或直接訪問：https://n88.zeabur.app/settings

### 問題 2：沒有「API」選項
**可能原因**：n8n 版本太舊或 API 功能未啟用
**解決方案**：
- 檢查 n8n 版本（頁面右下角）
- 如果版本 < 0.200.0，建議更新
- 或改用方案 B（手動匯入）

### 問題 3：無法創建 API Key
**可能原因**：權限不足
**解決方案**：
- 確認您是 owner 或 admin 身份
- 或聯繫管理員
